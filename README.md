# Checkpoint

Earn your rank, Pawn to King — a master's path that ranks your chess by piece.

A small, mobile-first chess-learning site. Plain static HTML/CSS/JS — no build step.

## Structure

- `index.html` — landing page
- `train/tactics.html` — Pawn Path tactics trainer (puzzles + badge)
- `train/kqk.html` — King + Queen vs King endgame trainer (perfect defence via the Lichess tablebase)

## Run locally

Just open `index.html` in a browser. (An internet connection is needed for the chess library and the endgame tablebase.)
